## :star2: About the Project

Convert any image into a Paint-by-the-Numbers template.

<u>Features</u>

- Selecting fidelity of the image
- Selecting the number of colours used to recreate the image
- Automatic number-to-colour assignment

## :floppy_disk: Data

Single image uploaded

## :palm_tree: Phases

- [✅] Phase 1: Summarising colours used in an image
- [✅] Phase 2: Drawing contours for image and colour assignment
- [✅] Phase 3: Testing complex architectures
- [✅] Phase 4: Web App development
- [] Phase 5: Deployment
